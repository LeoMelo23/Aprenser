## 📋 Sobre o projeto

O **Aprenser** é um exercício prático do curso de HTML e CSS da Udemy. O projeto simula a landing page de uma escola online com foco em cursos de programação, aplicando conceitos de estruturação semântica, estilização e layout responsivo.

---

## ✨ Funcionalidades

- 📱 **Layout responsivo** — adaptado para celulares, tablets e desktops
- 🎨 **Design moderno** — interface limpa e organizada para apresentação de cursos
- 🏫 **Estrutura de escola** — seções de cursos, apresentação e informações da instituição

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura e semântica da página |
| CSS3 | Estilização e responsividade |

---

## 📚 Conceitos praticados

- Semântica HTML5
- Flexbox  para layouts
- Design responsivo com Media Queries
- Organização e boas práticas de código
